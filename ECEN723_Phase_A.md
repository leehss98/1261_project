## Phase A Requirements

In Phase A, each **v-group** needs to finish the software coding of the vehicle system in the road described in the project overview. Each **i-group** needs to finish the software coding of signal control of the road system described in the overview. The code should be well tested and debugged.

The code of the **v-group** can assume signal control, while the code of the **i-group** can assume certain vehicle traffic. However, the v-group and i-group of the same team need to communicate with each other and build a protocol for information exchange, since the two codes will be integrated in **Phase B**.

## Report Contents

In the report, please include the following:

- the architecture of your code
- the algorithm of your code
- your source code
- examples of code test results

At the front of your report, please state your **team ID** and **associated group**.
